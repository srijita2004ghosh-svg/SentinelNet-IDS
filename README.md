# SentinelNet IDS

Enterprise AI-Powered Intrusion Detection System

## Features

- AI Threat Detection
- Isolation Forest & Random Forest Models
- Dynamic Risk Scoring
- Threat Intelligence Database
- Blockchain Audit Trail
- SOC Dashboard
- Interactive Visualizations
- CSV Export

## Technologies

- Python
- Streamlit
- Scikit-Learn
- Plotly
- Pandas
- Blockchain Hashing

## Run Locally

```bash
pip install -r requirements.txt
python -m streamlit run sentinel.py
```
